About Our Natural Gas Price Prediction Project

Project Overview

This project aims to develop a machine learning model to predict the price of natural gas based on historical data. By analyzing patterns and trends in historical price data, along with contextual factors such as the year, month, and day, the model provides accurate forecasts for future prices. The application is designed to help stakeholders in energy trading, production planning, and risk management make informed decisions.

Technology Stack

The project utilizes the following technologies:

  • Flask: A lightweight web framework for building the web application.
  • Scikit-Learn: A Python library used for developing and training the machine learning model.
  • NumPy: A library for numerical operations on data.
  • Pandas: A library for data manipulation and analysis.
  • HTML/CSS: For structuring and styling the web pages.

How It Works

The application allows users to input the year, month, and day to get a prediction of the natural gas price for that specific date. The machine learning model is trained on historical data and incorporates features such as the date to make accurate predictions. The system is designed to be user-friendly, providing an intuitive interface for users to interact with the model and receive predictions
